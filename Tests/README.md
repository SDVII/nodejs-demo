# Tests
    A NodeJs project for the sake of documenting NodeJs functionalities.

## Topics Covered
1. Testing functions
2. Testing routes
3. Asserting values
4. Using Spies

## Installation
1. Install NodeJS
2. Install TypeScript
3. In the project's root, run: ```npm install```
4. run: ```npm run test```
5. run: ```tsc --watch```
